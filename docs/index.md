# ✨ AINaming: AI-Powered Localization Naming

AINaming is a productivity tool for product designers and localization engineers that automatically suggests and applies consistent localization keys to Figma text layers.

### 📚 Documentation
- **[User Guide](./guide.html)**: Get started with AINaming quickly.
- **[Walkthrough for Designers](./WALKTHROUGH.html)**: Step-by-step tips.
- **[Terms of Service](./terms.html)**: Privacy and local data info.

### 🚀 Get Started
1. **Open the Plugin**: Hit `Settings` and enter your OpenAI or Anthropic API Key.
2. **Scan Selection**: Select frames in Figma to analyze.
3. **Confirm & Rename**: Set your Feature/Screen context and triage suggestions.
